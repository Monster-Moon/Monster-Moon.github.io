# Learning multiple quantiles with neural networks

This research is aim to solve the following inequality constrained problem:
$$
\min_{\beta_1, ..., \beta_T \in \mathbb{R}^{p}, \theta \in \Theta} \sum_{i = 1}^n \sum_{k=1}^K \rho_{\tau_k}(y_i - \textbf{z}(\textbf{x}_i, \theta)^\top\beta_k)\\
\text{subject to } \quad \textbf{z}(\textbf{x}, \theta)^\top\beta_{k-1} \leq \textbf{z}(\textbf{x}, \theta)^\top\beta_{k} \quad \text{ for all } ~~ \textbf{x} \in \mathbb{R}^p,
$$
where $\textbf{z}(\textbf{x}, \theta)$ is a feature map of a neural network with $\theta$, that is, an output of the last hidden layer for given $\textbf{x}$.
